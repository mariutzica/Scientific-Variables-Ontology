This folder contains the scripts necessary to clean up, standardize, and atomize (deconstruct) the CSN into its component parts.

atomize_CSN.py - script to break down each name and classify individual components
csn_utils.py - function library to generate standard name
make_CSN_list.py - combine the two CSN lists
fix_CSN_syntax.py - correct some syntactical errors, inconsistencies
