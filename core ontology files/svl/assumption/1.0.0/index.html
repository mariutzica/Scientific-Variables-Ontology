This is the documentation for the Assumption Ontology. All instances of the Assumption class are listed here. This class is a temporary catchall for restrictions, contextual information, or model settings that may be necessary in scientific modeling. The instances in this category will be merged into other classes and other ontological concepts in future versions of the ontology.

To read more about assumptions, please visit the CSDMS Standard Names documentation.
