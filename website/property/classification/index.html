classification

property

Definition: Classification is a broad concept that comprises the process of classifying, the set of groups resulting from classifying, and the assignment of elements to pre-established groups. Classifying is a fundamental concept and a part of almost all kinds of activities. Classification itself is an interdisciplinary field of study, with contributing disciplines including philosophy, biology, knowledge organization, psychology, statistics, and mathematics.
