weissenberg_number

property

Definition: The Weissenberg number (Wi) is a dimensionless number used in the study of viscoelastic flows. It is named after Karl Weissenberg. The dimensionless number compares the elastic forces to the viscous forces. It can be variously defined, but it is usually given by the relation of stress relaxation time of the fluid and a specific process time.
