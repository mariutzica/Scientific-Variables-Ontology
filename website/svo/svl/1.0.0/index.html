The Scientific Variables Lower Ontology (SVL) contains the instances, or individuals, of the Scientific Variables Ontology (SVO). Whereas the upper ontology (SVU) outlines the concept classes and defines the rules that determine how those atomistic concepts may be combined to create more complex concepts, SVL contains the declaration of various concept instances.
