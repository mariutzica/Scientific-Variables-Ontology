rotation

process

Definition: Rotation, or spin, is the circular movement of an object around a central axis. A two-dimensional rotating object has only one possible central axis and can rotate in either a clockwise or counterclockwise direction. A three-dimensional object has an infinite number of possible central axes and rotational directions.
