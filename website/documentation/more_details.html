Documenation for the ontology can be retrieved by visiting the appropriate URIs in the SVO namespace.

.

For detailed documentation about the overall ontology (SVO), please visit the URI for SVO.

Detailed documentation for the core upper level concepts in the ontology (SVU) can be found at the IRI for the upper ontology.

For documentation of all of the instances in the lower ontology (SVL), please visit the URI for SVL.

The code used to generate SVO 1.0.0 and encode the CSDMS standard names into the ontology format can be found in this GitHub repo.

The code used to power the term matching engine for SVO 1.0.0 can be found in this GitHub repo. This code is highly experimental and will be undergoing changes in the near future.
