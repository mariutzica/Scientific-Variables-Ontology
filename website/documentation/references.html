References

The Scientific Variables Ontology is designed with the best practices for the Semantic Web in mind. The following is a list of excellent resources that we used to learn about the Semantic Web and Ontologies. We highly recommend them to those interested in learning more!

The Semantic Web for the Working Ontologist: Effective Modeling in RDFS and OWL. Dean Allemang and Jim Hendler, 2011.

Concepts, Ontologies, and Knowledge Representation. Grega Jakus, Veljko Milutinović, Sanida Omerević, Sašo Tomažič, 2013.

The Elements of Knowledge Organization. Richard Smiraglia, 2014.
